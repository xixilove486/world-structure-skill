# Roadmap

This roadmap focuses on keeping `world-structure-skill` easy to install in Codex and easy to maintain.

## Phase 1: Stable Package Layout

Current focus:

- keep the installable payload inside `skills/world-structure/`
- keep `SKILL.md` lightweight and readable
- keep root docs focused on installation and repo guidance

## Phase 2: Better Skill Experience

Next improvements:

- polish templates
- add stronger example outputs
- add a short quick-start section for first-time users
- improve path references across the skill package

## Phase 3: Better Public Presentation

Potential additions:

- clearer GitHub front page
- a short Chinese intro and a short English intro
- one screenshot of the package layout or install flow
- a small FAQ for installation and usage

## Phase 4: Broader Reuse

Possible future work:

- domain-specific variants
- lighter and fuller editions of the skill
- translated versions
- curated companion skills if the project expands
