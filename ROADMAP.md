# Roadmap

This file outlines the suggested evolution path of `世界结构.skill`.

## Phase 1 — Stable Core
Current focus:
- keep `SKILL.md` lightweight
- keep the workflow stable
- make the output templates useful
- strengthen anti-weaponization guardrails

## Phase 2 — Better Examples
Needed:
- one organization example
- one relationship example
- one project bottleneck example
- one institution/rigidity example
- one renewal-attempt example

## Phase 3 — Better Reference Layer
Needed:
- polish `resources/world-model/`
- unify terminology
- reduce overlap across files
- add cross-references between long-form files

## Phase 4 — Better Public Presentation
Potential additions:
- cleaner GitHub front page
- repo badges
- short Chinese and English intros
- contribution examples
- FAQ for common misunderstandings

## Phase 5 — Optional Extensions
Possible future directions:
- compact “lite” version of the skill
- multilingual variants
- domain-specific adaptations
- richer example packs

## Things deliberately not prioritized
Not a priority right now:
- turning the repo into a giant philosophy archive
- forcing mathematical formalization
- pretending the skill predicts exact events
- expanding the vocabulary faster than the examples
