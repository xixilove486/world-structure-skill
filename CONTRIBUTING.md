# Contributing

Thank you for considering improvements to `世界结构.skill`.

This repository is small on purpose.  
The goal is not to make it bloated. The goal is to keep it sharp, reusable, and structurally disciplined.

## What kinds of contributions are welcome

Good contributions usually improve one of these:

- clarity
- structure
- guardrails
- example quality
- template usefulness
- conceptual consistency
- stage distinctions
- misuse prevention

Examples:
- clearer wording in `SKILL.md`
- better examples in `examples/`
- tighter definitions in `resources/core-concepts.md`
- stronger anti-weaponization language in `resources/guardrails.md`

## What kinds of contributions are not a fit

Please avoid changes that:

- turn the skill into a grand theory of everything
- make the language more dramatic but less precise
- add moralized labels to people
- remove uncertainty and alternative readings
- turn structural analysis into ideological preaching
- replace function with buzzwords

## Style expectations

Contributions should be:

- clear
- compact
- non-theatrical
- useful in real analysis
- aware of misuse risk

## Before proposing a change

Ask:

1. Does this make the skill more usable?
2. Does this reduce confusion?
3. Does this strengthen guardrails?
4. Does this preserve structural discipline?
5. Does this keep the skill lightweight?

If the answer to most of these is “no,” it probably does not belong.

## Preferred contribution areas

High priority:
- examples
- templates
- wording cleanup
- guardrail strengthening
- better stage distinctions

Lower priority:
- large theoretical expansion
- abstract metaphysics
- speculative universal claims

## Final note

This repository should stay sharper than it is grand.
