# 世界模型索引

本目录是 World Structure skill 的长参考层。

它包含：
- 供日常调用的**分段参考文件**

- 供来源、方法说明与验证状态使用的**参考 / 验证附录**

## 文件说明

- `00-index.md`  
  目录入口与导航说明。

- `01-model-positioning.md`  
  说明模型是什么、适合做什么、不适合做什么。

- `02-core-principles.md`  
  核心工作原则、使用纪律与防武器化姿态。

- `03-core-concepts.md`  
  核心结构概念，如锚点、角色、推力链、扰动、漂移、熵增与干预优先级。

- `04-evolution-stages.md`  
  阶段逻辑：形成、扩展、瓶颈、过渡、稳定、僵化、碎裂、更新尝试。

- `05-diagnostic-framework.md`  
  实际诊断流程。

- `06-intervention-rules.md`  
  修复逻辑与干预优先级。

- `07-boundaries-and-warnings.md`  
  模型边界、误用警告与使用限制。

- `08-extended-notes.md`  
  补充说明、边缘案例与扩展笔记。

- `13-references-and-validation.md`  
  参考来源、方法说明、理论对话与全局验证摘要。

- `world-model-full.md`  
  单文件完整长文版本。

## 说明

分段文件主要为了：
- 提升可读性
- 便于 GitHub 导航
- 便于分段维护
- 便于 skill 检索调用

完整长文则保留连续框架。

两者共同构成仓库的参考骨架。
