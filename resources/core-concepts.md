# Core Concepts

This file defines the minimal concept set used by `world-structure`.

## 1. System
A bounded but never perfectly sealed pattern of relationships, roles, memory, and pressures.

## 2. Anchor
The thing a system explicitly or implicitly orbits around: what it protects, pursues, or justifies itself through.

Possible states:
- clear
- split
- hollow
- replaced
- decorative

## 3. Constraint
A real limit on what the system can carry, coordinate, renew, or transform.

Examples:
- money
- time
- energy
- trust
- legitimacy
- memory
- coordination
- bandwidth
- institutional design

## 4. Layer
A meaningful level inside the system, such as core / middle / edge, or decision / execution / environment.

## 5. Role
A function rather than a personality label.

Typical role families:
- sustaining
- changing
- linking
- blocking
- extracting
- remembering
- buffering

## 6. Support Chain
The chain of things that keep the system running.

Examples:
- resources
- legitimacy
- emotional investment
- social trust
- institutional memory
- replacement pathways

## 7. Disturbance
Anything that repeatedly weakens the anchor, deforms transmission, erodes trust, or raises coordination cost.

Important:
Disturbance may be:
- a person
- a role
- an incentive
- a structural contradiction
- an environmental pressure
- a stage mismatch

## 8. Drift
When a system slowly moves away from its anchor, purpose, or functional center without openly naming the shift.

## 9. Entropy
The tendency of systems to become more costly, more ritualized, less adaptive, and less coherent unless renewed.

## 10. Stage
The rough developmental position of the system.

Suggested stage vocabulary:
- formation
- expansion
- bottleneck
- transition
- stabilization
- rigidity
- fragmentation
- renewal attempt

## 11. Core Contradiction
The deepest tension that generates multiple visible symptoms.

Examples:
- growth outrunning structure
- control outrunning trust
- extraction outrunning renewal
- identity outrunning reality
- memory collapsing under speed
- legitimacy declining while form remains

## 12. Intervention Priority
The first repair target with the highest structural leverage.
